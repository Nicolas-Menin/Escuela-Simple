# Sistema de Gestión Escolar

Sistema de gestión escolar desarrollado con **Python** y **FastAPI**, que permite administrar alumnos, docentes, cursos, materias e inscripciones.

---

## Tecnologías utilizadas

- Python
- FastAPI
- Uvicorn
- Pydantic
- SQLite (o base de datos relacional)
- Arquitectura en capas (Router + Schemas + DAO)
